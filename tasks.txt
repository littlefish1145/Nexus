# Task Dependencies

- Task 2 (事件通知) 依赖 Task 1 (SSE-C) — 事件载荷需要包含加密类型字段
- Task 4 (可观测性) 是其他所有 Task 的横切关注点，**优先于所有功能 Task** 落地
- Task 5 (KMS) 依赖 Task 16 (部署工具链) — KMS 需要 systemd / k8s secret 注入
- Task 7 (Raft) 依赖 Task 6 (配置校验) — Raft 配置需要 schema 校验保证正确性
- Task 8 (存储后端) 依赖 Task 7 (Raft) — 元数据共识后才能做数据分片
- Task 9 (备份) 依赖 Task 7 (Raft) — 增量备份需要基于共识日志的 lsn
- Task 10 (全文) 与 Task 11 (向量) 可并行
- Task 12 (断点续传) 依赖 Task 1 (SSE-C) — 续传完成时加密路径
- Task 13 (IAM) 依赖 Task 4 (可观测性) — 策略评估可观测
- Task 14 (SDK) 依赖 Task 1/2/12 — SDK 暴露这些功能
- Task 15 (文档) 与 Task 16 (部署) 持续进行，不阻塞主线
